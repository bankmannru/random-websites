Редактор уровней - Капуста 4: Путешествие во времени

Инструкции:

  • Выберите эпоху и тип объекта, который хотите разместить.
  • Для размещения платформы: нажмите и удерживайте левую кнопку мыши, затем перетащите, чтобы задать размер.
  • Для размещения других объектов: просто кликните в нужном месте.
  • Для порталов времени вам будет предложено ввести назначение (эпоху).
  • Сохраняйте свой прогресс регулярно!

Созданные уровни можно использовать в игре "Капуста 4: Путешествие во времени".
